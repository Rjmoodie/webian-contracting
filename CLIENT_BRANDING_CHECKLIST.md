### üé® 2. Visual Branding Assets

#### Logo Files
- [ ] **Primary Logo** (High Resolution)
  - **Format**: PNG with transparent background (preferred) or JPG

  - **File size**: Under 500KB for web performance
  - **Where used**: Navigation bar, email headers, printed materials

- [ ] **Logo Icon/Favicon** (Optional)
  - **Format**: PNG, ICO, or SVG
  - **Size**: 32x32px, 64x64px, 128x128px (multiple sizes recommended)
  - **Where used**: Browser tab icon, mobile home screen

- [ ] **Logo Variations** (Optional but recommended)
  - Light version (for dark backgrounds)
  - Dark version (for light backgrounds)
  - Monochrome version

#### Color Palette
- [ ] **Primary Brand Color**
  - Current default: `#755f52` (brown)
  - **Provide**: Hex code or color name
  - **Where used**: Main navigation, buttons, accents

- [ ] **Secondary/Accent Color**

  - **Provide**: Hex code or color name
  - **Where used**: CTAs, highlights, featured elements

- [ ] **Tertiary/Accent Color** (Optional)

  - **Provide**: Hex code or color name

- [ ] **Background Color**

### üé¨ 6. Portfolio Media Assets

#### Portfolio Items to Submit
For each portfolio item, provide:

- [ ] **Portfolio Item #1**
  - [ ] Title (e.g., "Corporate Event Photography - Kingston")
  - [ ] Description (2-3 sentences about the project)
  - [ ] Category: Photography / Videography / Audio
  - [ ] Media URL (direct link to image/video/audio file)
  - [ ] Thumbnail URL (optional, but recommended)
  - [ ] Event Type (e.g., Wedding, Corporate, Festival)
  - [ ] Parish/Location
  - [ ] Date of event
  - [ ] Talent/Photographer name (if applicable)
  - [ ] Mark as Featured? (Yes/No)

- [ ] **Portfolio Item #2**
  - [ ] (Same fields as above)

- [ ] **Portfolio Item #3**
  - [ ] (Same fields as above)

**Recommended**: Submit at least 6-12 portfolio items to start
- Mix of categories (photography, videography, audio)
- Mix of event types
- Mix of locations/parishes
- Include your absolute best work

#### Portfolio Media Requirements
- **Image Files**:
  - Format: JPG, PNG, or WebP
  - Size: Minimum 1200px width
  - File size: Under 2MB each (optimized)
  - Quality: High resolution, professional quality

- **Video Files**:
  - Format: MP4 (preferred) or WebM
  - Hosting: YouTube, Vimeo, or direct file hosting
  - Duration: 30 seconds to 5 minutes recommended
  - Quality: HD (1080p) or 4K

- **Audio Files**:
  - Format: MP3 or WAV
  - Quality: 128kbps minimum (MP3) or higher
  - Duration: 30 seconds to 5 minutes recommended

**üí° Tip**: Use a cloud storage service (Google Drive, Dropbox) or image hosting (Imgur, Cloudinary) and share direct links
